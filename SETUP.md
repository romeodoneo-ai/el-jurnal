# Журнал посещаемости СИС-12

## Быстрый старт (локально)

```bash
npm install
npm run dev
```

Откройте http://localhost:3000

---

## Деплой на Vercel (бесплатно)

### 1. Создайте аккаунт на Vercel
Зайдите на https://vercel.com и войдите через GitHub.

### 2. Загрузите проект на GitHub
```bash
git init
git add .
git commit -m "initial commit"
git remote add origin https://github.com/YOUR_USERNAME/el-jurnal.git
git push -u origin main
```

### 3. Подключите к Vercel
1. На Vercel нажмите "Add New Project"
2. Выберите ваш репозиторий `el-jurnal`
3. Нажмите "Deploy"
4. Через 1-2 минуты получите URL вида `https://el-jurnal.vercel.app`

### 4. Добавьте на рабочий стол телефона
1. Откройте URL на телефоне в Chrome/Safari
2. Chrome: меню (три точки) → "Добавить на главный экран"
3. Safari: кнопка "Поделиться" → "На экран Домой"

## Подключение к Google Sheets (опционально)

Если хотите, чтобы данные записывались напрямую в вашу Google Таблицу:

### 1. Откройте вашу Google Таблицу
### 2. Расширения → Apps Script
### 3. Замените содержимое файла Code.gs

Скопируйте код из файла `scripts/google-apps-script.js` в этом проекте.

### 4. Измените PIN
В начале скрипта найдите строку:
```javascript
const PIN = '1234';
```
Замените `1234` на свой PIN.

### 5. Разверните скрипт
1. Нажмите "Развернуть" → "Новое развёртывание"
2. Тип: **Веб-приложение**
3. Описание: "Журнал API"
4. Выполнять от имени: **Я** (ваш аккаунт)
5. Доступ: **Все** (Anyone)
6. Нажмите "Развернуть"
7. Подтвердите доступ (Google покажет предупреждение — нажмите "Дополнительно" → "Перейти к проекту")
8. Скопируйте URL развёртывания

### 6. Настройте приложение
1. Откройте приложение
2. Зайдите в Настройки (иконка шестерёнки)
3. Вставьте URL скрипта
4. Введите PIN
5. Сохраните

Теперь при нажатии "Сохранить" данные будут записываться и в localStorage, и в Google Таблицу.

## Структура проекта

```
src/
  app/
    page.tsx         — Главная: ввод посещаемости
    report/page.tsx  — Отчёт по месяцам
    settings/page.tsx — Настройки и бэкап
  lib/
    config.ts        — Список студентов и предметов
    storage.ts       — Работа с localStorage + генерация отчёта
    sync.ts          — Синхронизация с Google Sheets
    types.ts         — TypeScript типы
scripts/
  google-apps-script.js — Код для Google Apps Script
```

## Как изменить список студентов

Откройте `src/lib/config.ts` и отредактируйте массив `STUDENTS`.

## Как добавить/убрать предмет

В том же файле `src/lib/config.ts` отредактируйте массив `SUBJECTS`.
